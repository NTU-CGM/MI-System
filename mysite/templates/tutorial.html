※Imputation


MI-System accept the vcf.gz and plink format(.bed, .bim, .fam) files to upload, and allow users to upload through browser or url link(google drive)

※Notice that the files need to follow the rules:

  • Only contain single chromosome data
  • Coordinates version in hg19 (Except that using the hapmap3 as reference panels requires hg18)
  • Google drive url need to open the acesss right of the file

MI-System also provides two reference panels imputation:

  • Improved reference panels is the function that describes in IMPUTE2 ("Imputation with two phased reference panels") option
  • Merge reference panels is the function that describes in IMPUTE2 ("Imputation with two phased reference panels (merge reference panels)") option
